### О проекте:
Этот проект создан для обучения современной веб-разработке. Будет разработано веб-приложение Next Pizza(аналог Додо Пиццы). Здесь я изучаю полный цикл создания приложения - от фронтенда до бэкенда, используя актуальные технологии и инструменты.

### Стэк:
1. NextJS (Parallel Routes, Group Routes, Server Actions, API)
2. TypeScript
3. TailwindCSS + ShadCN
4. Prisma + PostgreSQL
5. NextAuth
6. React Hook Form + Zod
7. Zustand
8. react-use
9. nextjs-toploader
10. react-hot-toast
11. react-insta-stories
12. lucide-react
13. Resend
